Aiven documentation
===================

Aiven is a database-as-a-service platform for open source data solutions including Apache Kafka®, Apache Flink®, Apache Cassandra®, ClickHouse®, Grafana®, InfluxDB®, M3DB, MySQL, OpenSearch®, PostgreSQL®, and Redis®*.
